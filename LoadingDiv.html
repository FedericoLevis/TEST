1) JSU LoadingDiv

LoadingDiv API are designed to display a Loading Div during Long Operation. Many Options are available to choose Loading Gif, Title, Message, show/hide ElapsedSec,...
This features has a very simple API Interface in loadingDiv.js:
JSU File JSU API Item Description
jsu/core/loadingDiv.js loadingDivShow Show the Loading Div
loadingDivHide Hide (Destroy) the Loading Div
jsu/core/jsu.css CSS items used in loadingDiv.js LoadingDiv CSS: default background, gif, ..

We will descrive following cases:
  • Default LoadingDiv without using the Optional parameter objOpt
  • LoadingDiv Options using the Optional parameter objOpt to change Gif and to add Title, Cancel Button, ElapsedSec,...
  • Advanced LoadingDiv: For advanced LoadingDiv see SAMPLE_3 in LoadingDiv Sample

1.1) Default LoadingDiv

To use the default LoadingDiv you have to:
  • Call loadingDivShow without parameter
  • Call loadingDivHide to hide the loadingDiv when the Long operation has finished

1.2) LoadingDiv Options

You can use the Optional parameter objOpt to change Gif and to add Title, Show/Hide Cancel Button, ElapsedSec :
  • To show the loadingDiv before starting a Long operation: Call loadingDivShow setting the Desired objOpt
  • Call loadingDivHide to hide the loadingDiv when the Long operation has finished
In the Following Sample you can try some of the various LoadingDiv Options, setting the objOpt parameter with an UserFriendly Table:
